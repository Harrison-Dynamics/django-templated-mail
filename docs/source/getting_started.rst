Getting started
===============

Supported Python versions
-------------------------

* Python 2.7
* Python 3.4
* Python 3.5
* Python 3.6

Supported Django versions
-------------------------

* Django 1.8
* Django 1.10
* Django 1.11

Installation
------------

.. code-block:: bash

    $ pip install -U django-templated-mail

No need to add it to your ``INSTALLED_APPS``.
